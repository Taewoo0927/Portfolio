Utility Pole Monitoring Prototype · Sept 2025 – May 2026

Sensor-Based Utility Pole Monitoring Prototype

Designed and implemented a proof-of-concept pole-monitoring system that measures pole tilt and moisture using ESP32-C3 sensing nodes, an LIS3DH accelerometer, and an FDC1004 capacitance sensor. The system transmits readings through an ESP-based wireless mesh network to a fixed root node, then forwards data to a PC application for SQLite database storage, dashboard visualization, and offline map-based monitoring. The project integrated embedded firmware, PCB design, sensor calibration, wireless communication, and enclosure design for a battery-powered pole-mounted prototype.
